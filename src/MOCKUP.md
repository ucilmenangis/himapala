## Design Notes

### Color Scheme
- **Primary**: Dark Green (#2d5f4d) - Professional, nature-themed
- **Secondary**: Pastel Green (#c8e6d0) - Soft, calming
- **Background**: Light Green Tint (#f5f9f6) - Clean, fresh
- **Accent**: Medium Green (#a8d5ba) - Highlights

### Key Features
1. **Responsive**: Mobile-first design
2. **Clean Layout**: Spacious, easy to read
3. **Modern Cards**: Rounded corners, subtle shadows
4. **Icons**: Lucide React icons for consistency
5. **Smooth Scrolling**: Hero to Profile section on Beranda
6. **Glass Effect**: Transparent header with backdrop blur
7. **Grid Layouts**: 2-3 column responsive grids
8. **Images**: Unsplash for nature/hiking photos

### Typography
- Headers: Medium weight (500)
- Body: Normal weight (400)
- Clear hierarchy with defined sizes

### Interactive Elements
- Hover effects on cards (lift + shadow)
- Smooth transitions
- Active navigation highlighting
- Social media links open in new tabs
